CSS Selector Types

  1. Type Selectors - selects HTML by tag name
  2. Class Selectors - selects HTML by class name
  3. ID Selectors - selects HTML by ID name
            
              p {
                color: red;
              }

              .my-class-name {
                color: blue;
              }

              #my-id-name {
                color: purple;
              }

CSS Box Model

Image from internetingishard.com

The box model gives each element (both block and inline) four properties: content, padding, border, and margin. Content - the text, image or media content in the element. Padding - the space between the box's content and its border. Border - The line drawn around the content and padding. Margin - the space between an element and surrounding elements.
