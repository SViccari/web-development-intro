CSS: Type Selectors

Type Selectors are used to select an HTML element type.

Below is a example of using a CSS Type Selector. This will set the background-color for ALL paragraphs. 

            
              p {
                background-color: blue;
              }

Practice Type Selectors

In the .css file, add a CSS rule that:

  1. Selects all <p> tags
  2. Sets the background-color property to blue

But what if we only want one paragraph to be blue?

CSS: Class Selectors

  • Target HTML elements by using the HTML tag's class attribute.
  • Can be any name you choose
  • Written in lowercase letters
  • Words are separated by hypens

Below is an example of how to add a class attribute to an HTML tag. 

            
              

                Paragraph tag with a class attribute.

CSS: Class Selectors

A class selector is written in the CSS file by placing a period (.) before the class name. 

            
              .my-class-name {
                background-color: yellow;
              }

Practice Class Selectors

In the .html file...

Add a class attribute, with the name "my-class-name", to the first paragraph tag. 

            
              

                First paragraph.

Practice Class Selectors

In the .css file...

  1. Remove the CSS rule that selects all <p> tags.
  2. Add a CSS rule that selects the class name and set the background-color to yellow. 
            
              .my-class-name {
                background-color: yellow;
               }

Practice Class Selectors

CSS: ID Selectors

  • Target HTML elements by using the HTML tag's id attribute.
  • Are unique because each ID is only used once
  • Can be any name you choose
  • Written in lowercase letters
  • Words are separated by hypens

Below is an example of how to add an ID attribute to an HTML tag. 

            
              

                Paragraph tag with an ID attribute.

CSS: ID Selectors

An ID selector is written in the CSS file by placing a pound sign (#) before the ID name. 

            
              #my-id-name {
                background-color: purple;
              }

Practice ID Selectors

In the .html file...

  1. Remove the class attribute from the first paragraph
  2. Add the ID attribute, with the name "my-id-name", to the first paragraph. 
            
              

                First paragraph

Practice ID Selectors

In the .css file...

  1. Add a CSS rule that selects the ID attribute name and set the background-color to purple. 
            
              #my-id-name {
                background-color: purple;
              }

Recap of CSS Selector Types

  1. Type Selectors - selects HTML by tag name
  2. Class Selectors - selects HTML by class name
  3. ID Selectors - selects HTML by ID name
            
              p {
                color: red;
              }

              .my-class-name {
                color: blue;
              }

              #my-id-name {
                color: purple;
              }
