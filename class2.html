Block vs Inline HTML Elements

Image from https://codeburst.io

A block-level element always starts on a new line and takes up the full width of a page, from left to right. A block-level element can take up one line or multiple lines and has a line break before and after the element. An inline element does not cause a line break (start on a new line) and does not take up the full width of a page, only the space bounded by its opening and closing tag. It is usually used within other HTML elements.

What is CSS?

CSS stands for: Cascading Style Sheets

Cascading - rules cascade down from multiple sources.
Each rule can override a previous rule.

Style Sheets - refers to "style sheet language".
This language is used to describe the presentation of a document.

What can CSS do?

CSS can change colors, fonts, and position of content.
