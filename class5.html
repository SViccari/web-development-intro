List items - remove browser style

Goal: remove the style that is being applied by the browser to the list items or <li> tags.

Add the class "nav-list" to the <ul> tag

            
              .nav-list {
                list-style-type: none;
              }

Links - remove browser style

Goal: remove the style that is being applied by the browser to the links or <a>s tags.

Here, we are remove the browser default style for ALL links by targeting the "a" HTML tag. (ie: <a href="#">) 

            
              a {
                color: black;
                text-decoration: none;
              }

Horizontal Nav Bar

We live in the golden age of Flexbox. Flexbox replaces a number of CSS practices that were used to adjust where elements are positioned on a page.

Flexbox helps us create complex layouts with relatively little code.

Horizontal Nav Bar

To change the links to read left to right...

Add the following CSS:

            
              .nav-list {
                list-style-type: none;
                display: flex; /* new */
                flex-direction: row; /* new */
              }

Horizontal Nav Bar

Goal: Add space between the links.

            
              .nav-list li {
                padding-right: 40px;
              }

Nav - Background color

Goal: Apply a background color to the nav section.

            
              .nav-list {
                list-style-type: none;
                display: flex;
                flex-direction: row;
                background-color: #C8DCDE; /* new */
              }

Nav - Padding

Goal: Increase the height of the nav.

            
              .nav-list {
                list-style-type: none;
                display: flex;
                flex-direction: row;
                background-color: #C8DCDE;
                padding-top: 20px; /* new */
                padding-bottom: 20px; /* new */
              }

Nav - Margin

Goal: Remove unwanted white space.

            
              .body {
                margin: 0;
              }

              .nav-list {
                /* above properties collapsed */
                margin: 0;
              }

Set the font

Goal: Set the font type for all text.

            
              .body {
                margin: 0;
                font-family: Helvetica, Arial, sans-serif; /* new */
              }
Nav complete!
